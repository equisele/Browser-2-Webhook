Browser 2 Webhook

Browser 2 Webhook will allow you to send some information that you are browsing – as well as its title and meta description, if it has any – to a webhook from Automation tools such as Make, Zapier, Integrately, Airtable n8n, IFTTT and many more. In this way you can automate processes that are related to the web page you are visiting.

All you need is to replace the url you are seeing right now with one belonging to the automation tools mentioned above and save it by clicking the "Save" button.

Another major benefit is that Chrome's synced local storage is used; therefore, if you switch to another device, you will have the webhook url available.

Important: only following domains and paths are allowed to send POST requests:

  • https://hook.eu1.make.com/*
  • https://hook.us1.make.com/*
  • https://hook.integromat.com/*
  • https://hook.eu1.integromat.com/*
  • https://hook.us1.integromat.com/*
  • https://hooks.zapier.com/*
  • https://webhooks.integrately.com/*
  • https://*.hooks.n8n.cloud/*
  • https://hooks.airtable.com/workflows/*
  • https://maker.ifttt.com/trigger/*
  • https://cloud.yepcode.io/*
How to use it?

You have all the possibilities available in the contextual menu – right mouse click – being able to execute two tasks right now:

  • Send the tab url: title and meta description – if any – are also sent to webhook.
  • Send selected text to a webhook: just select a portion of text and send to webhook.
  • Send image to a webhook: right click over image to send it to webhook.
